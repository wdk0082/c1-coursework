Trainer Module
==============

Training, validation, and evaluation utilities for neural network models.

Overview
--------

The ``NNTrainer`` class handles the complete training workflow:

- Model instantiation and device placement
- Training loop with backpropagation
- Validation and model checkpointing
- Test set evaluation with multiple metrics
- Model saving and loading
- Training time and memory tracking
